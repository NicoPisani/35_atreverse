Project structure

First thank you for purchasing Resala - Charity & Fundraising  Template

to get the website working Make sure you upload the required files/folders listed below:

  • css    folder - containes stylesheets files 
  • js     folder - containes Javacripts Folder
  • img    folder - containes images 
  • fonts  folder - containes images 
  • style.css file 

   other folders

  • jade folder - containes Jade files , template was written in Jade Language to make it easy to edit and generate html , if you don't write in jade you can delete it
  • sass folder - containes sass files project was created in sass language to make it easy to edit and generate css

logo

Pleace your logo files in img folder. with name name logo.png

- if you want to edit logo height or width you can edit this line of code in html  <div id="logo"><a href="index-1.html"><img src="img/logo.png" alt=""></a></div>
- or in sass file _navbar.scss 

changing background

if you want to change section background  you can add one of the following pre-made classes 

.grey-bg - for making the background grey
.wh-bg - for making the background white
.dark-bg - for making the background dark
.violet-bg - for making the background violet
.red-bg - for making the background red
.teal-bg - for making the background teal
or you can style it as you want from style.css file

color changing

if you using  sass you can change the primary color of the project with only one line of code 

1- open variable.scss file
2- look for this line  $primary-color: #4ecdc4; , replace color with whatever color you want to edit
3 - it requires using sass if you dont know sass check this link
